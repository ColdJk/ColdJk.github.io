模型约定

Eloquent 也会假设每个数据表都有一个叫做 id 的主键字段。如果主键不是ID，也可以定义一个 $primaryKey 属性来重写这个约定。

默认情况下，Eloquent 会预计你的数据表中有 created_at 和 updated_at 字段。如果你不希望让 Eloquent 来自动维护这两个字段，可在模型内将 $timestamps 属性设置为 false
