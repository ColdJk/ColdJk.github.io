Windows Docker Desktop

设置默认wsl为ubuntu

  1. 确保分发以 WSL 2 模式运行。WSL可以在 v1 或 v2 模式下运行发行版，但 docker 只能在 WSL 2 上运行。
  1. Docker Desktop重新启动时，转到设置 > 资源 > WSL集成。
