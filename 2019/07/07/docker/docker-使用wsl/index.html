发表于 2019-07-07 更新于 2021-06-07 分类于 Docker

Windows Docker Desktop

设置默认wsl为ubuntu

  1. 确保分发以WSL 2模式运行。WSL可以在v1或v2模式下运行发行版。
  1. wsl -l -v 查看详情，将现有的Linux发行版升级到v2
  1. Docker Desktop重新启动时，转到设置 > 资源 > WSL集成。
